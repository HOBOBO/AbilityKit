## 📖 概念速查

### 核心类

| 类 | 职责 |
|------|------|
| `IRecordSession` | 会话接口（Facade） |
| `RecordContainer` | 可序列化数据容器 |
| `RecordTrack` | 单条录制轨道 |
| `RecordEvent` | 单条事件（FrameIndex + Type + Payload） |
| `IEventTrackWriter` | 事件写入接口 |
| `IEventTrackReader` | 事件读取接口 |
| `BasicReplayController` | 回放控制器 |
| `FixedStepReplayClock` | 固定步长时钟 |

### 内置轨道

| 轨道名称 | 用途 |
|----------|------|
| `inputs` | 玩家输入命令 |
| `state_hash` | 状态哈希采样 |
| `snapshots` | 世界快照 |
| `battle_log` | 战斗日志（自定义） |

### 状态

| 状态 | 含义 |
|------|------|
| `Running` | 回放中 |
| `Paused` | 已暂停 |
| `Finished` | 已结束 |
