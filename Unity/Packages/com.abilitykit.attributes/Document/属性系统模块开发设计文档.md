## 一、设计理念：为什么要做属性系统模块？

### 1.1 传统做法的问题

```
❌ 传统做法的问题：

1. 属性计算逻辑散落各处
   ┌─────────────────────────────────────────────────────────┐
   │ // 攻击力的计算到处都是：                               │
   │ float GetAttack() {                                   │
   │     return baseAttack + equipBonus + buffBonus;      │
   │ }                                                     │
   │                                                          │
   │ // 扣血时也要算：                                       │
   │ void TakeDamage(float dmg) {                          │
   │     var defense = baseDefense + buffDefense;          │
   │     var actualDmg = dmg * (1 - defense / 100);        │
   │     hp -= actualDmg;                                 │
   │ }                                                     │
   │                                                          │
   │ // 改一个地方要改N处，漏了就是Bug                      │
   └─────────────────────────────────────────────────────────┘

2. Buff/Debuff 管理混乱
   ┌─────────────────────────────────────────────────────────┐
   │ // 给单位加攻击力buff：                                │
   │ unit.attack += 100;                                   │
   │ unit.attackModifiers.Add(buffId, 100);                 │
   │                                                          │
   │ // 移除时要手动减：                                    │
   │ unit.attack -= unit.attackModifiers[buffId];           │
   │ unit.attackModifiers.Remove(buffId);                    │
   │                                                          │
   │ // 忘了减就是Bug                                       │
   │ // 减错了数值也是Bug                                    │
   └─────────────────────────────────────────────────────────┘

3. 属性依赖难以处理
   ┌─────────────────────────────────────────────────────────┐
   │ // 生命值 = 最大生命 + 额外生命                        │
   │ // 最大生命变了，生命值也要变                          │
   │ void SetMaxHp(float newMaxHp) {                       │
   │     var ratio = hp / maxHp;  // 先记录比例            │
   │     maxHp = newMaxHp;                                 │
   │     hp = maxHp * ratio;  // 再按比例恢复              │
   │ }                                                     │
   │                                                          │
   │ // 依赖多了就乱套                                      │
   └─────────────────────────────────────────────────────────┘

4. 性能问题
   ┌─────────────────────────────────────────────────────────┐
   │ // 每次获取属性都要重新算                              │
   │ float GetAttack() {                                   │
   │     var result = baseAttack;                          │
   │     foreach (var buff in allBuffs) {                  │
   │         if (buff.type == BuffType.Attack)            │
   │             result += buff.value;                    │
   │     }                                                 │
   │     return result;                                    │
   │ }                                                     │
   │                                                          │
   │ // 100个单位 × 50个属性 × 每帧多次访问 = 卡顿          │
   └─────────────────────────────────────────────────────────┘
```

### 1.2 属性系统模块的解决方案

```
✅ 属性系统的设计思路：

┌─────────────────────────────────────────────────────────────┐
│                       核心思想                              │
│                                                             │
│   把属性计算封装成"黑盒"，外部只管获取值                    │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                       │  │
│   │   外部代码：                                          │  │
│   │   ┌───────────────────────────────────────────┐    │  │
│   │   │  var attack = group.GetValue(AttackId);   │    │  │
│   │   └───────────────────────────────────────────┘    │  │
│   │   // 只需要调用，不需要知道怎么算                  │  │
│   │                                                       │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                       │  │
│   │   内部实现：                                          │  │
│   │   ┌───────────────────────────────────────────┐    │  │
│   │   │  Attack = (Base + Add) * (1 + Mul) + F   │    │  │
│   │   │                                               │    │  │
│   │   │  Add  = buff1.attack + buff2.attack + ... │    │  │
│   │   │  Mul  = buff1.mult + buff2.mult + ...      │    │  │
│   │   │  F    = buff1.finalAdd + ...               │    │  │
│   │   └───────────────────────────────────────────┘    │  │
│   │                                                       │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   好处：                                                    │
│   - 计算逻辑集中一处，不散落                               │
│   - Buff 用 Handle 管理，不会漏加漏减                       │
│   - 支持属性依赖，自动更新                                 │
│   - 脏标记 + 缓存，性能优化                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 核心理念

```
┌─────────────────────────────────────────────────────────────┐
│                    属性系统的三大承诺                          │
│                                                             │
│   1️⃣ 统一                                              │
│      所有属性计算集中管理，逻辑不散落                       │
│                                                             │
│   2️⃣ 高效                                              │
│      脏标记 + 缓存，只在必要时重新计算                      │
│                                                             │
│   3️⃣ 灵活                                              │
│      支持自定义公式、约束、依赖，可扩展                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 二、核心概念：从零理解属性系统

### 2.1 什么是"属性系统"？

**通俗解释**：属性系统是"计算器"，管理一个实体的所有属性值，并处理 buff/debuff 的叠加。

```
┌─────────────────────────────────────────────────────────────┐
│                    属性系统 = 计算器                        │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │  AttributeContext                                    │  │
│   │                                                       │  │
│   │  ┌───────────────────────────────────────────────┐  │  │
│   │  │                                               │  │  │
│   │  │  属性组 (AttributeGroup)                       │  │  │
│   │  │                                               │  │  │
│   │  │  ┌─────────────┐ ┌─────────────┐             │  │  │
│   │  │  │ 攻击力      │ │  生命值      │             │  │  │
│   │  │  │ 100 + 50    │ │  500 / 1000  │             │  │  │
│   │  │  │ = 150      │ │  = 500       │             │  │  │
│   │  │  └─────────────┘ └─────────────┘             │  │  │
│   │  │                                               │  │  │
│   │  │  ┌─────────────┐ ┌─────────────┐             │  │  │
│   │  │  │  防御力      │ │  移动速度    │             │  │  │
│   │  │  │  20 * 1.5   │ │  5 + 1      │             │  │  │
│   │  │  │  = 30      │ │  = 6        │             │  │  │
│   │  │  └─────────────┘ └─────────────┘             │  │  │
│   │  │                                               │  │  │
│   │  └───────────────────────────────────────────────┘  │  │
│   │                                                       │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │  修饰器 (Modifier)                                   │  │
│   │                                                       │  │
│   │  Add    = 直接加到基础值                             │  │
│   │  Mul    = 乘法叠加 (1 + m1 + m2 + ...)             │  │
│   │  FinalAdd = 最终加法                                │  │
│   │  Override = 强制覆盖                                │  │
│   │                                                       │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 关键名词解释

| 名词 | 通俗解释 | 类比 |
|------|----------|------|
| **AttributeDef** | 属性的"规格说明书" | 产品的参数表 |
| **AttributeId** | 属性的唯一标识 | 产品的型号 |
| **AttributeContext** | 实体的属性容器 | 一个人的所有证件 |
| **AttributeGroup** | 按组分类的属性集合 | 证件按类别归档 |
| **AttributeInstance** | 具体的属性值 | 一份证件 |
| **AttributeModifier** | 对属性的修改 | 证件上的批注/修改 |
| **ModifierHandle** | 修饰器的"遥控器" | 修改的撤销凭证 |
| **IAttributeFormula** | 属性计算公式 | 计算方法 |
| **IAttributeConstraint** | 属性约束器 | 取值范围限制 |

### 3.2 属性值计算流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                      属性值计算流程                                   │
│                                                                     │
│   获取属性值：group.GetValue(AttributeId)                          │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   1️⃣ 检查脏标记                                           │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                       │  │  │
│   │   │   Dirty == false?                                    │  │  │
│   │   │       │                                              │  │  │
│   │   │       ├─ Yes → 返回缓存值                           │  │  │
│   │   │       │                                              │  │  │
│   │   │       └─ No  → 进入计算                              │  │  │
│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                               │                             │  │
│   │                               ▼                             │  │
│   │   2️⃣ 获取修饰器累计值                                      │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                       │  │  │
│   │   │   Add     = mod1.val + mod2.val + ...              │  │  │
│   │   │   Mul     = 1 + mod1.val + mod2.val + ...         │  │  │
│   │   │   FinalAdd = mod1.val + ...                        │  │  │
│   │   │   Override = 存在则使用该值                        │  │  │
│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                               │                             │  │
│   │                               ▼                             │  │
│   │   3️⃣ 执行公式计算                                          │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                       │  │  │
│   │   │   value = (Base + Add) * Mul + FinalAdd            │  │  │
│   │   │                                                       │  │  │
│   │   │   if (HasOverride) value = Override                 │  │  │
│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                               │                             │  │
│   │                               ▼                             │  │
│   │   4️⃣ 应用约束器                                            │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                       │  │  │
│   │   │   value = Constraint.Apply(id, value);              │  │  │
│   │   │   // 例如: 限制生命值 >= 0 且 <= 最大生命           │  │  │
│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                               │                             │  │
│   │                               ▼                             │  │
│   │   5️⃣ 缓存结果                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                       │  │  │
│   │   │   CachedValue = value;                              │  │  │
│   │   │   Dirty = false;                                     │  │  │
│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │

│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
```

### 3.3 修饰器管理流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                      修饰器管理流程                                  │
│                                                                     │
│   添加修饰器：group.AddModifier(id, modifier)                      │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   1️⃣ 分配槽位                                             │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                       │  │  │
│   │   │   // 自由链表分配 O(1)                              │  │  │
│   │   │   if (_freeListHead != -1) {                        │  │  │
│   │   │       slotIndex = _freeListHead;                   │  │  │
│   │   │       _freeListHead = slots[slotIndex].NextFree;   │  │  │
│   │   │   }                                                  │  │  │
│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                               │                             │  │
│   │                               ▼                             │  │
│   │   2️⃣ 存储修饰器                                            │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                       │  │  │
│   │   │   slots[slotIndex] = {                              │  │  │
│   │   │       Modifier = modifier,                         │  │  │
│   │   │       Active = true,                               │  │  │
│   │   │       Handle = nextHandle++                        │  │  │
│   │   │   };                                                │  │  │
│   │   │                                                       │  │  │
│   │   │   handleMap[Handle] = slotIndex;                    │  │  │
│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                               │                             │  │
│   │                               ▼                             │  │
│   │   3️⃣ 更新累计值                                            │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                       │  │  │
│   │   │   switch (modifier.Op) {                            │  │  │
│   │   │       case Add:      accum.Add += modifier.Value;   │  │  │
│   │   │       case Mul:      accum.Mul += modifier.Value;   │  │  │
│   │   │       case FinalAdd: accum.FinalAdd += modifier.Value;│  │  │
│   │   │       case Override: accum.Override = modifier.Value; │  │  │
│   │   │                       accum.HasOverride = true;      │  │  │
│   │   │   }                                                  │  │  │
│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                               │                             │  │
│   │                               ▼                             │  │
│   │   4️⃣ 标记脏值                                              │  │
│   │   ┌─────────────────────────────────────────────────────┐  │  │
│   │   │                                                       │  │  │
│   │   │   slot.Dirty = true;                                │  │  │
│   │   │   return new ModifierHandle(handle);                │  │  │
│   │   │                                                       │  │  │
│   │   └─────────────────────────────────────────────────────┘  │  │
│   │                                                             │  │
│   └─────────────────────────────────────────────────────────────┘  │
│                                                                     │
│   移除修饰器：group.RemoveModifier(id, handle)                     │
│                                                                     │
│   ┌─────────────────────────────────────────────────────────────┐  │
│   │                                                             │  │
│   │   1️⃣ 查找槽位                                             │  │
│   │   slotIndex = handleMap[handle];                          │  │
│   │                                                             │  │
│   │   2️⃣ 标记为非活跃                                         │  │
│   │   slots[slotIndex].Active = false;                        │  │
│   │                                                             │  │
│   │   3️⃣ 重新计算累计值                                        │  │
│   │   recalc from all active modifiers                        │  │
│   │                                                             │  │

│   │   5️⃣ 回收槽位到自由链表                                    │  │
│   │   slots[slotIndex].NextFree = _freeListHead;              │  │
│   │   _freeListHead = slotIndex;                              │  │

│   └─────────────────────────────────────────────────────────────┘  │

└─────────────────────────────────────────────────────────────────────┘
```

### 4.4 AttributeModifier - 修饰器

**通俗解释**：Modifier 是"对证件的批注"，修改属性值。

```
┌─────────────────────────────────────────────────────────────┐
│                    AttributeModifier 结构                    │
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                                                       │  │

│   │   }                                                  │  │

│   │   ─────────────────────────────────────────────     │  │
│   │                                                       │  │
│   │   Add                                                 │  │
│   │   ─────────────────                                   │  │
│   │   直接加到基础值                                      │  │
│   │   value = (Base + Add) * Mul + FinalAdd              │  │
│   │   例: 基础100, Add=50 → (100+50)*1 = 150            │  │
│   │                                                       │  │
│   │   Mul                                                 │  │
│   │   ─────────────────                                   │  │
│   │   乘法叠加                                            │  │
│   │   value = (Base + Add) * (1 + Mul) + FinalAdd        │  │
│   │   例: 基础100, Mul=0.5 → (100+0)*(1+0.5) = 150      │  │
│   │   Mul1=0.2, Mul2=0.3 → (100)*(1+0.2+0.3) = 150      │  │
│   │                                                       │  │
│   │   FinalAdd                                           │  │
│   │   ─────────────────                                   │  │
│   │   最终加法，最后一步                                  │  │
│   │   value = (Base + Add) * Mul + FinalAdd              │  │
│   │   例: (100+50)*2 + 10 = 310                          │  │
│   │                                                       │  │
│   │   Override                                           │  │
│   │   ─────────────────                                   │  │
│   │   强制覆盖，优先级最高                               │  │
│   │   value = Override                                    │  │
│   │                                                       │  │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 五、设计模式总结

### 5.1 模块使用的设计模式

```
┌─────────────────────────────────────────────────────────────┐
│                    设计模式应用                              │
│                                                             │
│   1️⃣ Facade Pattern（门面模式）                            │
│      ─────────────────────────────────────                  │
│      Attributes 静态类提供简单易用的 API                      │
│      封装内部复杂的注册和查询逻辑                            │
│                                                             │
│      ┌─────────────────────────────────────────────────┐   │
│      │                                                   │   │
│      │   Attributes.Register(def);                     │   │
│      │   Attributes.Attr("name");                     │   │
│      │   Attributes.FreezeRegistry();                 │   │
│      │                                                   │   │
│      │   // 内部复杂的 Registry/Context 调用被隐藏   │   │
│      │                                                   │   │
│      └─────────────────────────────────────────────────┘   │
│                                                             │
│   2️⃣ Handle Pattern（句柄模式）                            │
│      ─────────────────────────────────────                  │
│      ModifierHandle 管理修饰器，支持 O(1) 添加/移除          │
│                                                             │
│      ┌─────────────────────────────────────────────────┐   │
│      │                                                   │   │
│      │   var handle = group.AddModifier(...);          │   │
│      │   // ...                                        │   │
│      │   group.RemoveModifier(handle);                 │   │
│      │                                                   │   │
│      └─────────────────────────────────────────────────┘   │
│                                                             │
│   3️⃣ Dirty Flag Pattern（脏标记模式）                     │
│      ─────────────────────────────────────                  │
│      值变化时标记脏，下次读取时重新计算                      │
│      避免频繁修改时的重复计算                                │
│                                                             │
│   4️⃣ Strategy Pattern（策略模式）                          │
│      ─────────────────────────────────────                  │
│      IAttributeFormula 支持自定义计算公式                    │
│      IAttributeConstraint 支持自定义约束                      │
│                                                             │
│   5️⃣ Object Pool Pattern（对象池模式）                     │
│      ─────────────────────────────────────                  │
│      修饰器槽位使用自由链表管理                              │
│      减少 GC 压力                                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 各模式解决的问题

| 模式 | 解决的问题 |
|------|------------|
| Facade | 简化 API，易于使用 |
| Handle | O(1) 修饰器管理 |
| Dirty Flag | 避免重复计算 |
| Strategy | 灵活的公式和约束 |
| Object Pool | 减少 GC 压力 |

## 六、使用指南

### 6.1 基础使用示例

```csharp
// 场景：角色属性系统

// 1. 初始化时注册属性
public static class GameAttributeIds
{
    public static AttributeId MaxHp;
    public static AttributeId Hp;
    public static AttributeId Attack;
    public static AttributeId Defense;
    public static AttributeId MoveSpeed;

    public static void Initialize()
    {
        MaxHp = Attributes.Register(new AttributeDef(
            name: "Battle.MaxHp",
            group: "Battle",
            defaultBaseValue: 1000f,
            constraint: new RangeAttributeConstraint(0f, float.MaxValue)
        ));

        Hp = Attributes.Register(new AttributeDef(
            name: "Battle.Hp",
            group: "Battle",
            defaultBaseValue: 1000f,
            constraint: new RangeAttributeConstraint(0f, float.MaxValue)
        ));

        Attack = Attributes.Register(new AttributeDef(
            name: "Battle.Attack",
            group: "Battle",
            defaultBaseValue: 100f
        ));

        Defense = Attributes.Register(new AttributeDef(
            name: "Battle.Defense",
            group: "Battle",
            defaultBaseValue: 20f
        ));

        // 冻结注册表
        Attributes.FreezeRegistry();
    }
}

// 2. 为角色创建属性上下文
public class Character
{
    private readonly AttributeContext _attrs;

    public Character()
    {
        _attrs = new AttributeContext();
        var group = _attrs.GetOrCreate("Battle");

        // 设置基础值
        group.SetBase(GameAttributeIds.MaxHp, 1000f);
        group.SetBase(GameAttributeIds.Hp, 1000f);
        group.SetBase(GameAttributeIds.Attack, 100f);
        group.SetBase(GameAttributeIds.Defense, 20f);
    }

    public float MaxHp => _attrs.GetOrCreate("Battle").GetValue(GameAttributeIds.MaxHp);
    public float Hp => _attrs.GetOrCreate("Battle").GetValue(GameAttributeIds.Hp);
    public float Attack => _attrs.GetOrCreate("Battle").GetValue(GameAttributeIds.Attack);
    public float Defense => _attrs.GetOrCreate("Battle").GetValue(GameAttributeIds.Defense);

    // 施放 buff
    public AttributeEffectHandle ApplyBuff(AttributeEffect buff)
    {
        return _attrs.ApplyEffect(buff, 0);
    }

    // 受伤
    public void TakeDamage(float damage)
    {
        var group = _attrs.GetOrCreate("Battle");
        var currentHp = group.GetValue(GameAttributeIds.Hp);
        var defense = group.GetValue(GameAttributeIds.Defense);

        // 伤害减免公式
        var reduction = defense / (defense + 100f);
        var actualDamage = damage * (1f - reduction);

        var newHp = Math.Max(0, currentHp - actualDamage);
        group.SetBase(GameAttributeIds.Hp, newHp);
    }
}

// 3. 使用
var hero = new Character();

// 加 buff
var powerBuff = new AttributeEffect(
    new AttributeEffect.Entry(GameAttributeIds.Attack,
        new AttributeModifier(AttributeModifierOp.Add, 50f)),
    sourceId: skillId
);

using (hero.ApplyBuff(powerBuff))
{
    Console.WriteLine($"攻击加成后: {hero.Attack}");  // 150
}

// buff 结束，自动移除
Console.WriteLine($"buff结束后: {hero.Attack}");  // 100
```

### 6.2 完整 Buff 系统示例

```csharp
// 定义 buff 效果
public static class BuffEffects
{
    // 力量祝福：攻击+50，生命上限+200
    public static AttributeEffect StrengthBlessing(int sourceId) =>
        new AttributeEffect(
            new AttributeEffect.Entry(GameAttributeIds.Attack,
                new AttributeModifier(AttributeModifierOp.Add, 50f, sourceId)),
            new AttributeEffect.Entry(GameAttributeIds.MaxHp,
                new AttributeModifier(AttributeModifierOp.Add, 200f, sourceId)),
            sourceId: sourceId
        );

    // 坚韧护盾：防御+30%
    public static AttributeEffect Shield(int sourceId) =>
        new AttributeEffect(
            new AttributeEffect.Entry(GameAttributeIds.Defense,
                new AttributeModifier(AttributeModifierOp.Mul, 0.3f, sourceId)),
            sourceId: sourceId
        );

    // 无敌：攻击归零
    public static AttributeEffect Invincible(int sourceId) =>
        new AttributeEffect(
            new AttributeEffect.Entry(GameAttributeIds.Attack,
                new AttributeModifier(AttributeModifierOp.Override, 0f, sourceId)),
            sourceId: sourceId
        );
}

// 角色使用 buff
var hero = new Character();

// 施放多个 buff
var buff1 = hero.ApplyBuff(BuffEffects.StrengthBlessing(skill1Id));
var buff2 = hero.ApplyBuff(BuffEffects.Shield(skill2Id));
var buff3 = hero.ApplyBuff(BuffEffects.Invincible(skill3Id));

Console.WriteLine($"攻击: {hero.Attack}");  // 0 (无敌覆盖)
Console.WriteLine($"防御: {hero.Defense}");  // 20 * 1.3 = 26

// 移除无敌
buff3.Dispose();
Console.WriteLine($"攻击: {hero.Attack}");  // 100 + 50 = 150

// 全部移除
buff1.Dispose();
buff2.Dispose();
```
