# EC 模块文档

本目录包含 EC（Entity-Component）实体组件模块的设计文档。

## 文档列表

- [EC模块开发设计文档](./EC模块开发设计文档.md) - 详细介绍 EC 模块的设计理念、核心架构和使用方法

## 概述

EC 模块是 Ability-Kit 中的轻量级实体组件系统，主要用于：

- 将历史代码统一管理到 EC 体系中
- 提供稳定的实体标识和生命周期管理
- 支持父子层级关系
- 通过事件系统驱动外部系统同步

> 注意：本模块不是完整意义上的高性能 ECS（如 archetype/chunk/query/system 管线），如果需要极致性能，建议引入成熟第三方 ECS 方案。
