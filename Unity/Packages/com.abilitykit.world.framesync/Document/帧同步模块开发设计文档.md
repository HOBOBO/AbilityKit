## 一、设计理念：什么是帧同步？

### 1.1 帧同步 vs 状态同步

在多人网络游戏开发中，有两种主要的网络同步方案：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        帧同步 vs 状态同步                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  【帧同步 (Frame Lockstep)】              【状态同步 (State Synchronization)】 │
│  ┌─────────────────────────────┐        ┌─────────────────────────────┐│
│  │  同步：玩家输入              │        │  同步：实体状态              ││
│  │                              │        │                              ││
│  │  流程：                      │        │  流程：                      ││
│  │  1. 玩家输入发送到服务器    │        │  1. 客户端发送输入到服务器    ││
│  │  2. 服务器广播输入到所有玩家 │        │  2. 服务器模拟游戏逻辑      ││
│  │  3. 所有客户端执行相同帧    │        │  3. 服务器广播状态快照      ││
│  │  4. 相同输入 → 相同结果    │        │  4. 客户端渲染服务器状态     ││
│  │                              │        │                              ││
│  │  优点：                      │        │  优点：                      ││
│  │  - 真正的一致性（确定性）    │        │  - 支持作弊检测             ││
│  │  - 录像录制简单（存输入）    │        │  - 网络延迟容忍度高         ││
│  │  - 服务器开销低（只转发）    │        │  - 实现相对简单             ││
│  │                              │        │                              ││
│  │  缺点：                      │        │  缺点：                      ││
│  │  - 任何不一致都会"蝴蝶效应" │        │  - 状态量大                 ││
│  │  - 需要保证100%确定性       │        │  - 延迟补偿复杂             ││
│  │  - 延迟容忍度低             │        │  - 录像量大                 ││
│  └─────────────────────────────┘        └─────────────────────────────┘│
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 帧同步的挑战

```
❌ 帧同步的三大挑战：

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  1. 确定性（Determinism）                                                │
│  ┌─────────────────────────────────────────────────────────────────┐  │
│  │  问题：相同输入必须产生相同结果                                   │  │
│  │                                                                  │  │
│  │  常见不一致来源：                                                │  │
│  │  - 浮点数精度差异（0.1 + 0.2 ≠ 0.3）                           │  │
│  │  - 随机数序列不同步                                              │  │
│  │  - 对象遍历顺序不确定                                           │  │
│  │  - 时间计算精度差异                                             │  │
│  │  - 物理引擎浮点误差                                             │  │
│  └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│  2. 延迟容忍（Latency Tolerance）                                        │
│  ┌─────────────────────────────────────────────────────────────────┐  │
│  │  问题：网络延迟导致输入到达时间不同                             │  │
│  │                                                                  │  │
│  │  传统方案：                                                      │  │
│  │  - Lockstep：等待所有玩家输入 → 延迟太高                        │  │
│  │  - 客户端预测：提前执行 → 需要回滚纠正                          │  │
│  └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│  3. 断线重连（Reconnection）                                             │
│  ┌─────────────────────────────────────────────────────────────────┐  │
│  │  问题：玩家断线重连后需要恢复到正确状态                         │  │
│  │                                                                  │  │
│  │  需求：                                                          │  │
│  │  - 保存历史帧状态快照                                            │  │
│  │  - 支持从任意帧恢复                                              │  │
│  │  - 增量同步 vs 全量同步                                         │  │
│  └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 2.2 帧时间接口

```csharp
namespace AbilityKit.Ability.FrameSync
{
    /// <summary>
    /// 帧时间接口，提供帧号与时间的转换
    /// </summary>
    public interface IFrameTime
    {
        /// <summary>
        /// 当前帧号
        /// </summary>
        FrameIndex Frame { get; }

        /// <summary>
        /// 上一帧到当前帧的时间增量（秒）
        /// </summary>
        float DeltaTime { get; }

        /// <summary>
        /// 累计时间（秒）
        /// </summary>
        float Time { get; }

        /// <summary>
        /// 将帧号转换为时间
        /// </summary>
        float FrameToTime(FrameIndex frame);

        /// <summary>
        /// 将时间转换为帧号
        /// </summary>
        FrameIndex TimeToFrame(float time);
    }

    /// <summary>
    /// 帧时间实现，支持固定时间步长
    /// </summary>
    public sealed class FrameTime : IFrameTime
    {
        private readonly float _fixedDelta;

        public FrameIndex Frame { get; private set; }
        public float DeltaTime { get; private set; }
        public float Time { get; private set; }

        public FrameTime(float fixedDeltaSeconds)
        {
            _fixedDelta = fixedDeltaSeconds;
            Frame = FrameIndex.Zero;
            DeltaTime = 0f;
            Time = 0f;
        }

        public void StepTo(FrameIndex nextFrame, float deltaTime)
        {
            Frame = nextFrame;
            DeltaTime = deltaTime;
            Time += deltaTime;
        }

        public void Reset(FrameIndex frame, float time, float fixedDelta)
        {
            Frame = frame;
            Time = time;
            _fixedDelta = fixedDelta;
        }

        public float FrameToTime(FrameIndex frame)
        {
            return frame.Value * _fixedDelta;
        }

        public FrameIndex TimeToFrame(float time)
        {
            return new FrameIndex((int)(time / _fixedDelta));
        }
    }
}
```

### 2.3 帧驱动接口

```csharp
namespace AbilityKit.Ability.FrameSync
{
    /// <summary>
    /// 帧驱动接口，驱动整个游戏世界按帧推进
    /// </summary>
    public interface IFrameDriver
    {
        /// <summary>
        /// 当前帧号
        /// </summary>
        FrameIndex Frame { get; }

        /// <summary>
        /// 执行一帧
        /// </summary>
        /// <param name="deltaTime">帧时间增量</param>
        void Step(float deltaTime);
    }

    /// <summary>
    /// 世界管理器帧驱动实现
    /// </summary>
    public sealed class WorldManagerFrameDriver : IFrameDriver
    {
        private readonly IWorldManager _worlds;
        private FrameIndex _frame;

        public FrameIndex Frame => _frame;

        public WorldManagerFrameDriver(IWorldManager worlds)
        {
            _worlds = worlds;
        }

        public void Step(float deltaTime)
        {
            // 1. 驱动所有世界 Tick
            _worlds.Tick(deltaTime);

            // 2. 递增帧号
            _frame = _frame.Next();
        }
    }
}
```

### 3.1 为什么需要回滚预测？

### 3.2 回滚状态提供者

```csharp
namespace AbilityKit.Ability.FrameSync.Rollback
{
    /// <summary>
    /// 可回滚状态提供者接口
    /// 实现此接口的系统状态将被纳入回滚范围
    /// </summary>
    public interface IRollbackStateProvider
    {
        /// <summary>
        /// 唯一键值，用于注册表索引
        /// </summary>
        int Key { get; }

        /// <summary>
        /// 导出指定帧的当前状态
        /// </summary>
        /// <param name="frame">帧号</param>
        /// <returns>序列化后的状态数据</returns>
        byte[] Export(FrameIndex frame);

        /// <summary>
        /// 导入历史状态
        /// </summary>
        /// <param name="frame">帧号</param>
        /// <param name="payload">序列化状态数据</param>
        void Import(FrameIndex frame, byte[] payload);
    }

    /// <summary>
    /// 世界状态 Hash 提供者接口
    /// 用于对账时计算和比较状态
    /// </summary>
    public interface IWorldStateHashProvider
    {
        /// <summary>
        /// 尝试获取指定帧的状态 Hash
        /// </summary>
        bool TryGetHash(FrameIndex frame, out WorldStateHash hash);
    }

    /// <summary>
    /// 状态 Hash 值
    /// 使用 CRC32/FNV 等快速哈希算法
    /// </summary>
    public readonly struct WorldStateHash
    {
        public readonly uint Value;

        public WorldStateHash(uint value)
        {
            Value = value;
        }

        public static bool operator ==(WorldStateHash left, WorldStateHash right)
            => left.Value == right.Value;

        public static bool operator !=(WorldStateHash left, WorldStateHash right)
            => left.Value != right.Value;

        public override bool Equals(object obj)
            => obj is WorldStateHash other && Value == other.Value;

        public override int GetHashCode() => Value.GetHashCode();
    }
}
```

### 3.3 回滚注册表

```csharp
namespace AbilityKit.Ability.FrameSync.Rollback
{
    /// <summary>
    /// 回滚注册表
    /// 收集和管理所有 IRollbackStateProvider
    /// </summary>
    public sealed class RollbackRegistry
    {
        private readonly List<IRollbackStateProvider> _providers = new();
        private readonly Dictionary<int, IRollbackStateProvider> _byKey = new();
        private bool _sealed = false;

        /// <summary>
        /// 注册状态提供者
        /// </summary>
        public void Register(IRollbackStateProvider provider)
        {
            if (_sealed)
                throw new InvalidOperationException("Cannot register after seal");

            if (_byKey.ContainsKey(provider.Key))
                throw new ArgumentException($"Duplicate key: {provider.Key}");

            _providers.Add(provider);
            _byKey[provider.Key] = provider;
        }

        /// <summary>
        /// 密封注册表，之后不能再注册
        /// </summary>
        public void Seal()
        {
            _sealed = true;
        }

        /// <summary>
        /// 根据键获取提供者
        /// </summary>
        public bool TryGet(int key, out IRollbackStateProvider provider)
        {
            return _byKey.TryGetValue(key, out provider);
        }

        /// <summary>
        /// 获取所有提供者
        /// </summary>
        public IReadOnlyList<IRollbackStateProvider> Providers => _providers;
    }
}
```

### 3.4 回滚快照

```csharp
namespace AbilityKit.Ability.FrameSync.Rollback
{
    /// <summary>
    /// 世界回滚快照
    /// 包含所有可回滚系统的状态
    /// </summary>
    public struct WorldRollbackSnapshot
    {
        /// <summary>
        /// 快照对应的帧号
        /// </summary>
        public FrameIndex Frame;

        /// <summary>
        /// 各系统的状态数据
        /// </summary>
        public byte[][] States;

        public WorldRollbackSnapshot(FrameIndex frame, byte[][] states)
        {
            Frame = frame;
            States = states;
        }
    }

    /// <summary>
    /// 回滚快照环形缓冲区
    /// 保存历史帧的快照，用于回滚
    /// </summary>
    public sealed class RollbackSnapshotRingBuffer
    {
        private readonly int _capacity;
        private readonly WorldRollbackSnapshot[] _snapshots;
        private readonly bool[] _hasSnapshot;
        private int _head = 0;  // 写入位置
        private FrameIndex _minFrame;

        public RollbackSnapshotRingBuffer(int capacity)
        {
            _capacity = capacity;
            _snapshots = new WorldRollbackSnapshot[capacity];
            _hasSnapshot = new bool[capacity];
            _minFrame = FrameIndex.Invalid;
        }

        /// <summary>
        /// 存储快照
        /// </summary>
        public void Store(FrameIndex frame, WorldRollbackSnapshot snapshot)
        {
            int index = frame.Value % _capacity;

            _snapshots[index] = snapshot;
            _hasSnapshot[index] = true;

            if (!_minFrame.IsValid || frame.Value < _minFrame.Value)
                _minFrame = frame;

            _head = (index + 1) % _capacity;
        }

        /// <summary>
        /// 尝试获取指定帧的快照
        /// </summary>
        public bool TryGet(FrameIndex frame, out WorldRollbackSnapshot snapshot)
        {
            if (!_minFrame.IsValid || frame.Value < _minFrame.Value)
            {
                snapshot = default;
                return false;
            }

            int index = frame.Value % _capacity;
            if (_hasSnapshot[index] && _snapshots[index].Frame == frame)
            {
                snapshot = _snapshots[index];
                return true;
            }

            snapshot = default;
            return false;
        }

        /// <summary>
        /// 清除所有快照
        /// </summary>
        public void Clear()
        {
            Array.Clear(_hasSnapshot, 0, _capacity);
            _minFrame = FrameIndex.Invalid;
        }
    }
}
```

```csharp
        /// <param name="computeHash">状态 Hash 计算函数</param>
        public void TickPredicted(
            FrameIndex nextFrame,
            IReadOnlyList<PlayerInputCommand> inputs,
            Func<FrameIndex, WorldStateHash> computeHash)
        {
            // 1. 提交输入到世界
            _inputSink.Submit(nextFrame, inputs);

            // 2. 保存输入历史
            _inputs.Store(nextFrame, inputs);

            // 3. 执行世界 Tick
            _world.Tick(/* fixedDelta */);

            // 4. 保存回滚快照
            _rollback.CaptureAndStore(nextFrame);

            // 5. 计算并记录状态 Hash
            var hash = computeHash(nextFrame);
            _reconciler.RecordPredictedHash(nextFrame, hash);

            PredictedFrame = nextFrame;
        }

        /// <summary>
        /// 处理权威状态 Hash
        /// </summary>
        /// <returns>是否需要回滚</returns>
        public bool OnAuthoritativeStateHash(FrameIndex frame, WorldStateHash authoritative)
        {
            return _reconciler.OnAuthoritativeHash(frame, authoritative);
        }

        /// <summary>
        /// 处理回滚请求
        /// </summary>
        public void HandleRollbackRequested(FrameIndex rollbackFrame)
        {
            // 1. 尝试恢复到目标帧
            if (!_rollback.TryRestore(rollbackFrame))
                return;

            // 2. 从目标帧重演到预测帧
            for (var f = rollbackFrame.Next(); f.Value <= PredictedFrame.Value; f = f.Next())
            {
                if (!_inputs.TryGet(f, out var inputs))
                    continue;

                _inputSink.Submit(f, inputs);
                _world.Tick(/* fixedDelta */);
                _rollback.CaptureAndStore(f);

                var hash = /* compute hash */;
                _reconciler.RecordPredictedHash(f, hash);
            }

            ConfirmedFrame = rollbackFrame;
        }
    }
}
```

```csharp
    /// <summary>
    /// 客户端预测对账器
    /// 比较预测 Hash 和权威 Hash，触发回滚
    /// </summary>
    public sealed class ClientPredictionReconciler
    {
        private readonly WorldStateHashRingBuffer _predicted;
        private FrameIndex _lastConfirmed = FrameIndex.Invalid;

        /// <summary>
        /// 当需要回滚时触发
        /// </summary>
        public event Action<FrameIndex> OnRollbackRequested;

        public ClientPredictionReconciler(int bufferCapacity)
        {
            _predicted = new WorldStateHashRingBuffer(bufferCapacity);
        }

        /// <summary>
        /// 记录预测帧的 Hash
        /// </summary>
        public void RecordPredictedHash(FrameIndex frame, WorldStateHash hash)
        {
            _predicted.Store(frame, hash);
        }

        /// <summary>
        /// 处理权威 Hash
        /// </summary>
        /// <returns>是否触发了回滚</returns>
        public bool OnAuthoritativeHash(FrameIndex frame, WorldStateHash authoritative)
        {
            // 1. 清理已确认帧之前的缓存
            if (_lastConfirmed.IsValid && frame.Value > _lastConfirmed.Value)
            {
                // 可以清理 _lastConfirmed 之前的记录
            }

            // 2. 尝试获取预测 Hash
            if (!_predicted.TryGet(frame, out var predicted))
            {
                // 没有预测记录，忽略
                return false;
            }

            // 3. 比较 Hash
            if (predicted == authoritative)
            {
                // Hash 匹配，状态一致
                _lastConfirmed = frame;
                return false;
            }

            // 4. Hash 不匹配，触发回滚
            // 回滚到上一个确认帧
            OnRollbackRequested?.Invoke(_lastConfirmed.IsValid ? _lastConfirmed : FrameIndex.Zero);
            return true;
        }
    }
```

```csharp
namespace AbilityKit.Ability.FrameSync.Rollback
{
    /// <summary>
    /// 输入历史环形缓冲区
    /// 保存历史帧的输入，用于回滚重演
    /// </summary>
    public sealed class InputHistoryRingBuffer
    {
        private readonly int _capacity;
        private readonly FrameIndex[] _frames;
        private readonly PlayerInputCommand[][] _inputs;
        private readonly bool[] _hasInput;
        private FrameIndex _minFrame;

        public InputHistoryRingBuffer(int capacity)
        {
            _capacity = capacity;
            _frames = new FrameIndex[capacity];
            _inputs = new PlayerInputCommand[capacity][];
            _hasInput = new bool[capacity];
        }

        /// <summary>
        /// 存储指定帧的输入
        /// </summary>
        public void Store(FrameIndex frame, IReadOnlyList<PlayerInputCommand> inputs)
        {
            int index = frame.Value % _capacity;

            _frames[index] = frame;
            _inputs[index] = inputs is PlayerInputCommand[] arr ? arr : inputs.ToArray();
            _hasInput[index] = true;

            if (!_minFrame.IsValid || frame.Value < _minFrame.Value)
                _minFrame = frame;
        }

        /// <summary>
        /// 尝试获取指定帧的输入
        /// </summary>
        public bool TryGet(FrameIndex frame, out PlayerInputCommand[] inputs)
        {
            if (!_minFrame.IsValid || frame.Value < _minFrame.Value)
            {
                inputs = null;
                return false;
            }

            int index = frame.Value % _capacity;
            if (_hasInput[index] && _frames[index] == frame)
            {
                inputs = _inputs[index];
                return true;
            }

            inputs = null;
            return false;
        }

        /// <summary>
        /// 清除所有记录
        /// </summary>
        public void Clear()
        {
            Array.Clear(_hasInput, 0, _capacity);
            _minFrame = FrameIndex.Invalid;
        }
    }
}
```

### 3.8 回滚协调器

```csharp
namespace AbilityKit.Ability.FrameSync.Rollback
{
    /// <summary>
    /// 回滚协调器
    /// 协调多个系统回滚
    /// </summary>
    public sealed class RollbackCoordinator
    {
        private readonly RollbackRegistry _registry;
        private readonly RollbackSnapshotRingBuffer _buffer;

        public RollbackCoordinator(RollbackRegistry registry, int bufferCapacity)
        {
            _registry = registry;
            _buffer = new RollbackSnapshotRingBuffer(bufferCapacity);
        }

        /// <summary>
        /// 捕获并存储当前状态
        /// </summary>
        public bool CaptureAndStore(FrameIndex frame)
        {
            var providers = _registry.Providers;
            var states = new byte[providers.Count][];

            for (int i = 0; i < providers.Count; i++)
            {
                states[i] = providers[i].Export(frame);
            }

            var snapshot = new WorldRollbackSnapshot(frame, states);
            _buffer.Store(frame, snapshot);
            return true;
        }

        /// <summary>
        /// 尝试恢复到指定帧
        /// </summary>
        public bool TryRestore(FrameIndex frame)
        {
            if (!_buffer.TryGet(frame, out var snapshot))
                return false;

            Restore(snapshot);
            return true;
        }

        /// <summary>
        /// 执行恢复操作
        /// </summary>
        public void Restore(WorldRollbackSnapshot snapshot)
        {
            var providers = _registry.Providers;

            for (int i = 0; i < providers.Count; i++)
            {
                if (snapshot.States[i] != null)
                {
                    providers[i].Import(snapshot.Frame, snapshot.States[i]);
                }
            }
        }
    }
}
```

```csharp
    /// 帧同步驱动模块
    /// 负责服务器的帧推进和帧广播
    /// </summary>
    public sealed class FrameSyncDriverModule : IHostRuntimeModule
    {

        private HostRuntimeOptions _options;
        private IWorldManager _worlds;
        private Dictionary<WorldId, WorldSession> _sessions = new();

        public void Install(HostRuntime runtime, HostRuntimeOptions options)
        {
            _runtime = runtime;
            _options = options;
            _worlds = runtime.Worlds;

            // 注册 Hook
            _options.Hook.PreTick.Register(OnPreTick, priority: 0);
            _options.Hook.PostTick.Register(OnPostTick, priority: 0);
        }

        private void OnPreTick(float deltaTime)
        {
            foreach (var session in _sessions.Values)
            {
                // 1. 提交所有待处理的输入
                foreach (var (playerId, inputs) in session.PendingInputs)
                {
                    var sink = session.World.Services.Resolve<IWorldInputSink>();
                    sink.Submit(session.PendingFrame, inputs);
                }
                session.PendingInputs.Clear();

                // 2. 执行世界 Tick
                session.World.Tick(deltaTime);
            }
        }

        private void OnPostTick(float deltaTime)
        {
            foreach (var session in _sessions.Values)
            {
                // 1. 收集快照
                var snapshotProvider = session.World.Services.Resolve<IWorldStateSnapshotProvider>();
                snapshotProvider.TryGetSnapshot(session.Frame, out var snapshot);

                // 2. 构建帧包
                var packet = new FramePacket(
                    session.WorldId,
                    session.Frame,
                    GetAllInputs(session),
                    snapshot);

                // 3. 广播到所有客户端
                _runtime.Broadcast(new FrameMessage(packet));
            }
        }
```

```csharp
            _options = options;

            _options.Hook.PreTick.Register(OnPreTick, priority: 0);
            _options.Hook.PostTick.Register(OnPostTick, priority: 0);
        }

        private void OnPreTick(float deltaTime)
        {
            foreach (var ctx in _contexts.Values)
            {
                // 1. 计算预测窗口（EWMA 平滑）
                int window = CalculatePredictionWindow();

                // 2. 推进确认帧（权威输入）
                while (ctx.ConfirmedFrame.Value < ctx.AuthoritativeFrame.Value)
                {
                    ctx.ConfirmedFrame = ctx.ConfirmedFrame.Next();
                    ApplyAuthoritativeInput(ctx, ctx.ConfirmedFrame);
                }

                // 3. Replay 模式：重演到确认帧
                if (ctx.Mode == ReplayMode.Replaying)
                {
                    ReplayToFrame(ctx, ctx.ConfirmedFrame);
                }

                // 4. 预测推进
                while (ctx.PredictedFrame.Value < ctx.ConfirmedFrame.Value + window)
                {
                    var next = ctx.PredictedFrame.Next();
                    ApplyLocalInput(ctx, next);
                    ctx.PredictedRunner.TickPredicted(next, ctx.LocalInputs[next], ComputeHash);
                }
            }
        }

        private void OnPostTick(float deltaTime)
        {
            foreach (var ctx in _contexts.Values)
            {
                // 捕获快照
                ctx.Rollback.CaptureAndStore(ctx.PredictedFrame);

                // 记录 Hash
                var hash = ComputeHash(ctx.PredictedFrame);
                ctx.Reconciler.RecordPredictedHash(ctx.PredictedFrame, hash);
            }
        }

        /// <summary>
        /// 处理权威状态 Hash
        /// </summary>
        public void OnAuthoritativeStateHash(WorldId worldId, FrameIndex frame, WorldStateHash hash)
        {
            if (!_contexts.TryGetValue(worldId, out var ctx))
                return;

            if (ctx.Reconciler.OnAuthoritativeHash(frame, hash))
            {
                // 需要回滚
                ctx.Mode = ReplayMode.Replaying;
                ctx.Rollback.HandleRollbackRequested(frame);
                ctx.ConfirmedFrame = frame;
                ctx.Mode = ReplayMode.Normal;
            }
        }
    }
}
```

## 七、设计总结

### 7.1 核心优势

| 特性 | 说明 |
|------|------|
| **确定性** | 所有系统必须实现 `IRollbackStateProvider` 保证同输入同结果 |
| **可追溯** | 环形缓冲区保存历史帧，支持断线重连和录像回放 |
| **平滑预测** | EWMA 动态调整预测窗口，适应不同网络条件 |
| **模块化** | 回滚系统和预测系统解耦，易于扩展 |
| **Host 扩展** | 通过 `IHostRuntimeModule` 灵活集成帧同步能力 |

### 7.2 扩展点

1. **自定义 Hash 算法**：实现 `IWorldStateHashProvider`
2. **自定义回滚策略**：扩展 `RollbackCoordinator`
3. **预测调优**：通过 `IClientPredictionTuningControl` 动态调整
4. **多世界支持**：每个世界独立的回滚上下文

### 7.3 注意事项

- 帧同步要求所有参与系统都是确定性的
- 避免使用 `Dictionary` 遍历、`Guid.NewGuid()` 等不确定操作
- 浮点数比较需要考虑精度容差
- 环形缓冲区容量需要根据网络延迟和回滚需求合理设置
