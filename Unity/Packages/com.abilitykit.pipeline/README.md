# com.abilitykit.pipeline

本包提供 Pipeline 运行时与相关的图结构/同步能力，并包含一套**以 Run 为中心**的调试数据模型（LiveRegistry + Trace），用于在 Play Mode 下观察运行中的 pipeline。
