# AbilityKit Orleans Server

This folder contains a minimal Orleans template for local development and future integration testing with the networking module.

## Projects

- `AbilityKit.Orleans.Contracts`: Grain interfaces and DTOs.
- `AbilityKit.Orleans.Grains`: Grain implementations.
- `AbilityKit.Orleans.Host`: Silo host.

## Run

- Build and run `AbilityKit.Orleans.Host`.
